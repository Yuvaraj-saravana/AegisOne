# SecurityOps Unified - Cybersecurity Platform

## Project Goal
Build a unified SOC (Security Operations Center) dashboard that integrates:
- SIEM (Wazuh) for log/alert aggregation
- Vulnerability scanning (OpenVAS)
- Threat intelligence (MISP)
- Ticketing system for incidents
- AI assistant for security queries
- Email security scanner
- Deepfake detection

## Tech Stack
- Backend: Python FastAPI
- Frontend: React + Tailwind CSS
- Database: PostgreSQL + Redis
- Queue: Celery + Redis
- Containerization: Docker Compose
- AI: Ollama (local LLM)
